# 质数的个数

## 1. 普通的质因数分解

对于一般的数，直接循环进行分解：

~~~java
public static int countPrimer(int n){
    int cnt = 0;

    for(int i = 2; i * i <= n; ++i){
        while(n % i == 0){
            cnt++;
            n /= i;
        }
    }

    if(n > 1){
        cnt++;
    }

    return cnt;
}
~~~

> 上述代码的解释：
>
> 为什么对呢，我们知道每一个数都可以由**唯一的一串质数**的乘积得到，我们可以从最小的质数 2 开始，判断 i 是不是 n 的约数，如果是，循环将 n 中的 i 全部约掉，并同时进行计数。最后如果 n > 1，表明 n 也是一个质数。
